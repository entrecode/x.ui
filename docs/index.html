x.ui {{version}}{{dependancy}}

x.ui is a framework developed for a fast forward unicorn frontend-development workflow. the unicon workflow is focused on modularity and structure. From the smallest inline-element, through more complex macro elements and even more advanced partial elements to templates - to pages. Everything is highly adaptable and flexible.

globals

Globals are basic HTML elements like headings, typeface, lists, form elements, images and tables. Globals are the smallest particles in an unicorn driven frontend-development workflow.

button globals/_btn.scss

there are three basic button styles which should be used to to emphasis or weaken functions. Throughout x.ui there are three steps of importance. The default style always reached with th default class btn, the minor style by an additional modifier class btn_minorand the superior style by the additional modifier btn_super
